# CS-341 Project
## Rendering 3-D Fractal Images via Raytracing
### Instructions
* Install `make`, `cmake`, and `g++`.
```shell
$ mkdir build
$ cd build
$ cmake ..
$ make
$ cd ..
$ build/raytracer examples/[name].json

# If you are using macOS, to open the rendered image:
$ open out/[name].png
```
